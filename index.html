Les blogs

Ajouter un blog

Les critères pour proposer l’ajout d’un blog :

  • Le blog n’est pas à l’abandon
  • Le blog est au moins partiellement en français
  • Le blog n'est pas hébergé sur une plateforme qui peut lui imposer ses choix (réseaux sociaux, Medium, Substack…)
  • Alex a envie d'inclure le blog dans son blogroll

Pour proposer un blog, envoyez un mail à blogroll@alexsirac.com avec pour objet "Proposition : le nom du blog proposé". En corps du mail, mettez le lien du blog, une description en une phrase, et le lien du flux RSS du blog si vous le connaissez. Vous pouvez inclure toute autre remarque si vous le souhaitez.

Il est apprécié, mais pas obligatoire, de faire un lien vers blogroll.fr depuis votre blog.
